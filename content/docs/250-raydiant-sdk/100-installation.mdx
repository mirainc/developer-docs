title: Installation
excerpt: How to install the Raydiant SDK.
---

To make it easy to add to any app, you can install the Raydiant SDK from [NPM](https://www.npmjs.com/) or include a global script.

## NPM Install

```bash
npm install @raydiant/sdk
# or yarn
yarn add @raydiant/sdk
```

```js
import { runtime, devTools } from '@raydiant/sdk';
```

## Include Global Script

```html
<script src="https://cdn.raydiant.com/lib/raydiant-sdk@0.9.0.min.js"></script>
```

```js
const { runtime, devTools } = window.Raydiant;
```

List the menus your On-Brand Menu API key has access to.

## Request

Set the `X-API-Key: [API Key]` header.

```bash
curl --request GET 'https://menu-api.raydiant.com/v1/menus' \
     --header 'X-API-Key: [API Key]'
```

## Response

```json
[
  {
    "id": "cf499066-a415-4dc3-bcce-e5482357b512",
    "name": "My Menu",
    "locations": {
      "name": "Location Name",
      "restaurnName": "Restaurant Name"
    }
  }
]
```

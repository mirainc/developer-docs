
## Deploy

To deploy your own Weather app you’ll need to first create an account at [OpenWeather](https://home.openweathermap.org/) to get your API key.

Click the below “Deploy" button to deploy the Weather app to your own Vercel account.

<a href="https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fmirainc%2Fsimple-weather-app&env=OPENWEATHERMAP_API_KEY&envDescription=OpenWeather%20API%20key&envLink=https%3A%2F%2Fhome.openweathermap.org%2F&redirect-url=https%3A%2F%2Fdevelopers.raydiant.com%2Fdocs%2Fget-started%2Fsample-apps%2Fsimple-weather">
  <img src="https://vercel.com/button" alt="Deploy with Vercel" />
</a>

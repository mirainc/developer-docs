Add a _button group_ input to your app to allow users choose from a list of 2-3 options.

## Options

Add options to your _button group_ input and set a default option. The input ID in the URL template for your _button group_ input is replaced with the selected optionâ€™s value.

## Validation

The _button group_ input supports the following validation options:

- **Required:** Prevent your app from rendering until an input value is provided.
